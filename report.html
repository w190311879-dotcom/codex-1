Report & Removal

投诉与下架通道

如果你是权利人、当事人、监护人、授权代理人，或发现疑似违法和高风险内容，请通过本页面提交投诉。

疑似未成年人、非自愿内容、偷拍视频、隐私泄露、名人或 AI 深伪内容会被优先处理。

请提供这些信息

  1. 具体 URL，不接受只写域名或模糊描述。
  2. 问题类型：版权、隐私、非自愿、疑似未成年、冒名、AI 深伪、其他。
  3. 你的身份：权利人、当事人、监护人、代理人或举报者。
  4. 说明材料：截图、授权证明、身份证明或能支持投诉的事实。
  5. 可联系邮箱，方便我们反馈处理结果或要求补充材料。

优先处理类型

  • 疑似儿童性虐待材料或未成年人性化内容。
  • 非自愿亲密图像、偷拍视频、报复性发布和隐私泄露。
  • 名人或普通人的 AI 换脸、深伪、冒名亲密内容。
  • 明确版权侵权且 URL 定位清楚的内容。

提交邮箱

建议邮件主题格式：Urgent Removal Request - URL 或 DMCA Takedown Request - URL。
